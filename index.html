Welcome to the Pronoun Closet! Here, you can try out different combinations of names and pronouns and rate them to see which ones you are most comfortable with. No data is stored by the website, but please note that your browser may store some of the names and pronouns you try in cache.

This was a very quick and simple project and is meant to be used short-term for trying different combinations of things. As such, there are some glitches, for example it is possible to rate the same name/pronoun set multiple times and have it appear multiple times in the list. Since patching this would require a lot of rewriting and this site is meant to be of short-term use anyway, I decided to leave it as is.

Using this site is simple. Fill in the fields with the name and pronouns you would like to try out and click submit to see a sample sentence using your chosen name and pronouns. You can click submit again to see a different sample sentence if desired. Afterwards, you can rate how you feel about the set of name/pronouns and it will appear in an automatically sorted list. You can click any of the items in this list to try the name/pronoun set again.

The singular/plural switch is for words like is/are and was/were, which depend on the pronoun used. An example of the use of a singular pronoun would be, "He is a good friend." An example of a plural pronoun would be, "They are a good friend." You can set this option depending on the pronouns you are trying so that the words used in the sample sentences are grammatically correct.
